# Wiki — audit_pyside

Este diretório contém a versão versionada das páginas propostas para a Wiki do projeto `audit_pyside`.

A Wiki do GitHub usa arquivos Markdown por página. Os arquivos deste diretório foram nomeados para poderem ser copiados para o repositório wiki (`audit_pyside.wiki.git`) quando a aba Wiki estiver habilitada.

## Páginas criadas

| Arquivo | Finalidade |
|---|---|
| `Home.md` | Página inicial da Wiki |
| `Arquitetura-e-Pipeline.md` | Visão de arquitetura, camadas, dependências e fio de rastreabilidade |
| `Catalogo-de-Tabelas-e-Campos.md` | Explicação de cada tabela documentada e de seus campos canônicos |
| `Instalacao-e-Execucao.md` | Como instalar dependências, rodar testes e iniciar a aplicação |
| `Operacao-e-Scripts.md` | Runbook dos scripts e geração de artefatos |
| `Desenvolvimento-e-Contribuicao.md` | Fluxo de contribuição, branches, PRs, testes e qualidade |
| `Publicacao-da-Wiki.md` | Como publicar estes arquivos na aba Wiki do GitHub |

## Fonte canônica

A documentação técnica detalhada continua em `docs/README.md` e nos documentos específicos de `docs/`.
A Wiki funciona como porta de entrada rápida para instalação, operação, arquitetura, catálogo de tabelas e contribuição.

## Regra de manutenção

Sempre que uma página da Wiki for alterada, mantenha a versão correspondente neste diretório para preservar histórico, revisão por PR e rastreabilidade.
